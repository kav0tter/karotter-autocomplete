# Flash Writer

[Karotter](https://karotter.com) 向け AI インライン自動補完拡張機能。

入力中にゴーストテキストで補完候補が表示されます。**Tab** で確定、**Esc** で却下。

## 特徴

- **ゴーストテキスト** — カーソル位置に薄いテキストで補完候補を表示
- **FIM (Fill-in-the-Middle)** — カーソルの前後を `prompt` / `suffix` として送信し、文脈を理解した補完を実現
- **発火タイミング設定** — すべての変更 / 入力のみ / 空白・改行のみ / カーソル移動 から選択可能
- **ビルトインプリセット** — Codestral / OpenAI / Gemini を選ぶだけで使える
- **カスタムプリセット** — API設定を保存して切り替え
- **使用状況トラッキング** — リクエスト数・トークン使用量を確認
- **ダークモード** — OS設定に自動追従
- **軽量** — 依存ライブラリなし、バニラJS、Manifest V3

## インストール

1. リポジトリをダウンロードまたはクローン
2. `chrome://extensions` を開く
3. 右上の **デベロッパーモード** を有効化
4. **パッケージ化されていない拡張機能を読み込む** → `chrome/` ディレクトリを選択
5. ツールバーの拡張機能アイコンをクリック → サイドパネルでAPIを設定

## 設定

ツールバーアイコンからサイドパネルを開きます。

| 設定 | デフォルト | 説明 |
|---|---|---|
| 有効/無効 | ON | マスタートグル |
| エンドポイントURL | — | FIM API のエンドポイント |
| API Key | — | APIのBearerトークン |
| モデル | `codestral-latest` | モデル識別子 |
| 遅延 | `1.0` 秒 | 入力停止から補完リクエストまでの時間 |
| 発火タイミング | すべての変更 | 補完を発火するタイミング |
| カーソル移動で発火 | OFF | 矢印キーやクリックでも発火 |
| システムプロンプト | SNS向け | FIMプロンプトに埋め込む指示文 |

### ビルトインプリセット

| プリセット | エンドポイント | モデル |
|---|---|---|
| Codestral (Mistral) | `codestral.mistral.ai/v1/fim/completions` | `codestral-latest` |
| OpenAI | `api.openai.com/v1/chat/completions` | `gpt-4o-mini` |
| Gemini (Google) | `generativelanguage.googleapis.com/v1beta/openai/chat/completions` | `gemini-2.0-flash` |

プリセットを選ぶとエンドポイントURL・モデル・システムプロンプトが自動入力されます。API Keyを入力して保存してください。

## 仕組み

```
┌─────────────────────────────────────┐
│  textarea                           │
│  ┌───────────────────────────────┐  │
│  │ 今日はいい天気だ░ね           │  │  ← ░ がカーソル位置
│  │              ↑ ゴーストテキスト│  │  ← 薄い灰色の補完候補
│  └───────────────────────────────┘  │
│                 [↹ Tab で確定]       │  ← ヒントバッジ
└─────────────────────────────────────┘
```

1. ユーザーが `<textarea>` に入力
2. 設定された遅延の後、カーソル位置でテキストを分割してバックグラウンドに送信
3. バックグラウンドワーカーが FIM API を呼び出し（`prompt` = カーソル前、`suffix` = カーソル後）
4. レスポンスをテキストエリアに重ねた透過オーバーレイに表示
5. **Tab** で補完をカーソル位置に挿入、**Esc** で却下

### リクエストフロー

```
content.js                    background.js                 API
─────────                     ────────────                  ───
onInput() → デバウンス
         ──── sendMessage ───→ handleAutocomplete()
                               カーソル位置でテキスト分割
                               FIMリクエストを構築
                               ──── fetch ────→  POST /completions
                               ←─── response ────
         ←── { data } ─────── 補完テキストを抽出
showSuggestion()
```

## プロジェクト構成

```
chrome/
├── manifest.json      Manifest V3 設定
├── background.js      サービスワーカー — API通信
├── content.js         コンテンツスクリプト — DOM注入・オーバーレイ
├── content.css        ゴーストテキスト・ヒントバッジのスタイル
├── popup.html         サイドパネルUI
├── popup.js           設定・プリセット・統計のロジック
└── icons/             拡張機能アイコン (16/48/128px)
```

## 開発

ビルド不要。`chrome/` 内のファイルを編集し、`chrome://extensions` から拡張機能をリロードしてください。
